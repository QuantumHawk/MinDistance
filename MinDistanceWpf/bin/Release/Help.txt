Для того, чтобы начать работу с программой:
1. Расположите нажатием левой кнопки мыши Точки (Вершины)
на читом поле, расположенном в нижней части экрана
2. Соедините вершины: сначала выберите первую вершину, нажатием
левой кнопки мыши,программа предложит выбрать вторую, нажмите на вторую.

Когда граф будет построен, выберите один из алгоритмов
Для генетического алгоритма первоначально необходимо задать Кол-во цветов и Размер популяции.

Для очистки таблицы результатов, нужно нажать по нему правой кнопкой мыши.

Для полной очистки следует нажать кнопку Clean.

Для сброса результатов раскраски следует нажать кнопку Restart.
